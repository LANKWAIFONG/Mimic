# Mimic One

Fundamental factor scoring and point-in-time backtesting for equity research, built on Piotroski-style signals.

## Stack

- Next.js (App Router) + Tailwind — frontend & API routes
- Supabase — auth & database
- Stripe — subscriptions
- Vercel — hosting & cron jobs

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
